## Удаление и откладывание

<span style="color:red">**git stash push** </span> - отложить изменения. Откладывает изменения, чтобы вы, например, могли срочно перейти к другой задаче. 

Пример транскрипции:

```bash=
git stash push
```
Чтобы отложить только часть изменений, используйте 
```bash=
git stash --patch
```
Вернуть отложенные изменения
```bash=
git stash pop
```
