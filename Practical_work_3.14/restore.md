## Удаление и откладывание

<span style="color:red">**git restore *[Название_файла]***</span> - отменить изменения, не добавленные в индекс — git restore [Название_файла]. Удалит изменения в одном файле.

Пример транскрипции:

```bash=
git restore [name file]
```
Чтобы удалить изменения во всех файлах, используйте
```bash=
git restore :/
```
