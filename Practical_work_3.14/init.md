## Работа с репозиторием

<span style="color:red">**git init**</span> - создает пустой репозиторий на вашем компьютере. Команда создаёт в текущей директории новую поддиректорию с именем .git, содержащую все необходимые файлы репозитория 

Пример транскрипции:

```bash=
git init
```
