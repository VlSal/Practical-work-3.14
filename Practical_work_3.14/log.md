## Работа с изменениями

<span style="color:red">**git log**</span> - gосмотреть историю коммитов. Выводит список всех коммитов. У этой команды есть разные опции, самая используемая из них —
```bash=
--oneline
``` 
Она показывает хеш в укороченном формате, ветку, в которой сделан коммит, а также текст коммита. Чтобы использовать эту опцию (как и любую другую), нужно добавить её после команды.

Пример транскрипции:

```bash=
git log--oneline
```
