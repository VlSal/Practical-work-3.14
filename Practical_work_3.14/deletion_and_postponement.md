## Удаление и откладывание

<span style="color:red">**git clean**</span> - удалить лишнее. Команда «наводит чистоту» — удаляет неотслеживаемые файлы из рабочего каталога.

Пример транскрипции:

```bash=
git clean
```

<span style="color:red">**git reset --hard** </span>- отменить изменения, добавленные в индекс — `git reset --hard`. Возвращает изменения из индекса и полностью их отменяет.

Пример транскрипции:

```bash=
git reset --hard
```

<span style="color:red">**git restore *[Название_файла]***</span> - отменить изменения, не добавленные в индекс — `git restore` [Название_файла]. Удалит изменения в одном файле.

Пример транскрипции:

```bash=
git restore [name file]
```
Чтобы удалить изменения во всех файлах, используйте
```bash=
git restore :/
```

<span style="color:red">**git revert *[хеш]*** </span> - удалить коммит — `git revert [195dfb0]`. Вместо [195dfb0] указывается хеш коммита, его можно узнать с помощью команды `git log`.

Пример транскрипции:

```bash=
git revert [195dfb0]
```

<span style="color:red">**git stash push** </span> - отложить изменения. Откладывает изменения, чтобы вы, например, могли срочно перейти к другой задаче. 

Пример транскрипции:

```bash=
git stash push
```
Чтобы отложить только часть изменений, используйте 
```bash=
git stash --patch
```
Вернуть отложенные изменения
```bash=
git stash pop
```
