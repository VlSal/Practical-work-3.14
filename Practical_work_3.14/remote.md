## Работа с репозиторием

<span style="color:red">**git remote add *[имя_удаленного_репозитория] [адрес]*** </span> - добавлениt связанных удаленных репозиториев. Удалённые репозитории это версии вашего IT-проекта, сохраненные в сети (локальной или интернет).

Пример транскрипции:

```bash=
git remote add origin https://github.com/vvvvvv/example
```
