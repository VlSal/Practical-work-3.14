## Работа с изменениями

<span style="color:red">**git pull** </span> - подтянуть изменения. Подтягивает в локальный репозиторий последнюю версию проекта. Вызов этой команды сотрёт все незафиксированные изменения.

Пример транскрипции:

```bash=
git pull
```
