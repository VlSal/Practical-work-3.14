## Работа с репозиторием

<span style="color:red">**git init** </span> - создает пустой репозиторий на вашем компьютере. Команда создаёт в текущей директории новую поддиректорию с именем .git, содержащую все необходимые файлы репозитория 

Пример транскрипции:

```bash=
git init
```


<span style="color:red">**git clone *[ссылка_на_удалённый_репозиторий]*** </span> - cклонировать удалённый репозиторий. Проект появится в директории, где вы находились в момент клонирования.Клонировать можно как локальный (находящийся на вашем компьютере), так и удалённый (находящийся в сети) репозиторий.

Пример транскрипции:

```bash=
git clone https://github.com/vvvvvv/example
```


<span style="color:red">**git remote add *[имя_удаленного_репозитория] [адрес]*** </span> - добавлениt связанных удаленных репозиториев. Удалённые репозитории это версии вашего IT-проекта, сохраненные в сети (локальной или интернет).

Пример транскрипции:

```bash=
git remote add origin https://github.com/vvvvvv/example
```
