## Удаление и откладывание

<span style="color:red">**git reset --hard** </span>- Отменить изменения, добавленные в индекс — git reset --hard. Возвращает изменения из индекса и полностью их отменяет.

Пример транскрипции:

```bash=
git reset --hard
```
